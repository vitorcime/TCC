# TCC-1

Trabalho de conclusão de curso desenvolvido em linguagem Python, com orientação do professor Juliano Henrique Foleiss. O projeto é o desenvolvimento de uma rede neural, usando a biblioteca tensorflow, para o reconhecimento de eventos sonoros em um áudio. O dataset foi obtido através do desafio DCASE de 2018 (http://dcase.community/challenge2018/index)
